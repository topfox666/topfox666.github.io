偶然从一位叫刘三刀的大神博客里看到一篇文章：

蓝眼睛的人

一座岛上有各种各样眼睛的人，但没有人知道自己眼睛的颜色，岛上也没有镜子，村民们无法自己知道自己眼睛的颜色。这座岛有一个奇怪的规矩：村民们不准讨论关于眼睛颜色的事情，如果有人知道了自己眼睛的颜色，那他必须在知道信息的当天自杀。

一天，村里来了个贤者，他认识了村里所有的人，走之前他说：“你们之中有蓝眼睛的人”。如果村子里一共有100个人，有5个蓝眼睛的人，在贤者说完这句话之后，会发生什么？

不妨模拟一下n个蓝眼睛会发生什么

java

顺便复习一下非常非常基础的几个方法的使用

1、生成随机数

  • 实例化Random：new Random()

借助java.util.Random类来产生一个随机数发生器，也是最常用的一种，构造函数有两个，Random()和Random(long seed)。第一个就是以当前时间为默认种子，第二个是以指定的种子值进行。产生之后，借助不同的语句产生不同类型的数。

种子就是产生随机数的第一次使用值,机制是通过一个函数,将这个种子的值转化为随机数空间中的某一个点上,并且产生的随机数均匀的散布在空间中。以后产生的随机数都与前一个随机数有关。

public static void main(String[] args)
{    
　　Random r = new Random(1);
　　for(int i=0 ; i<5 ;  i++)
　　{
　　　　int ran1 = r.nextInt(100);
　　　　System.out.println(ran1);
　　}
}
产生的五个数为85,88,47,13,54
  • 调用java.lang.Math包下的函数：Math.random()

而第二种方法返回的数值是[0.0,1.0）的double型数值，由于double类数的精度很高，可以在一定程度下看做随机数，借助（int）来进行类型转换就可以得到整数随机数了，例子中用的就是这个方法。

  • 利用当前时间（不常用）：currentTimeMillis()
public static void main(String[] args)
    {    
        int max=100,min=1;
        long randomNum = System.currentTimeMillis();  
        int ran3 = (int) (randomNum%(max-min)+min);  
        System.out.println(ran3);
        }

至于第三种方法虽然不常用，但是也是一种思路。方法返回从1970年1月1日0时0分0秒（这与UNIX系统有关）到现在的一个long型的毫秒数，取模之后即可得到所需范围内的随机数。

2、Java迭代器

  • Iterable 实现这个接口的集合对象支持迭代，是可以迭代的。实现了这个可以配合foreach使用。
//Iterable JDK源码
//可以通过成员内部类，方法内部类，甚至匿名内部类去实现Iterator
public interface Iterable
{
   Iterator iterator();
  • Iterator 迭代器，提供迭代机制的对象，具体如何迭代是这个Iterator接口规范的。
  • Iterator接口：包含三个方法：hasNext，next，remove、remove一般很少用到。

3、Arrays.asList()

该方法是将数组转化成List集合的方法

好用是好用，但是有几个注意事项：

  • 该方法适用于对象型数据的数组（String、Integer…）
  • 该方法不建议使用于基本数据类型的数组（byte,short,int,long,float,double,boolean）
  • 该方法将数组与List列表链接起来：当更新其一个时，另一个自动更新
  • 不支持add()、remove()、clear()等方法

那么问题来咯：

statement output
Arrays.asList(new Integer[] { 1, 2 }) (I)
Arrays.asList(new String[] { “a”, “b” }) (II)
Arrays.asList(new int[] { 1, 2 }) (III)
Arrays.asList(new String[] { “a”, “b” }, “c”) (IV)

(I)-(IV)分别多少呢?
